# Remnant Library
Game engine for the Game Boy Advance built on top of the [butano](https://github.com/GValiente/butano) library.
(Currently in)capable of top-down action RPG, side-scrolling platformer and visual novel functionality.
Used in [Minigame Investigator](https://github.com/JK0JK/minigame-investigator). In a way this is a way to make the project's core open-source without fear of copyright infringement.

# Warning
This project does NOT build programs for Linux, Windows macOS, or any platform other than the GBA. While there is code that allows for that code to be easily implemented in the future, I have no interest of working on it, as that is not the focus of this project. You are free to add your own implementation/substitution of the Butano engine for the purposes of allowing for multi-platform exports.
