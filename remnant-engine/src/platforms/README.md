# Platform-specific implementations
As one may be able to see from [core.h](/remnant-engine/include/platforms/core.h) and others, you can create your own implementations of the [butano engine](https://GValiente/butano)'s specific libraries for different platforms.

**This is NOT planned future functionality for the Remnant Engine!!!!** It is simply an easy way for you to make your own implementations if you feel like it.

I will never touch a windows or macos computer again if i can help it, and neither should you. But in case you do, consider creating your own implementation for each of these.

For homogeneity, keep the namespace "bn" because you're adapting the butano engine.
